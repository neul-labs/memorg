# Getting Started

This section will help you get up and running with Memorg quickly.

## Prerequisites

- Python 3.11 or higher
- OpenAI API key

## Quick Overview

Memorg can be used in two ways:

1. **As a Python Library** - Integrate directly into your applications
2. **As a CLI Tool** - Interactive command-line interface for standalone use
